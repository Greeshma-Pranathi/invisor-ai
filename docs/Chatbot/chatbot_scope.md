# Chatbot Scope & Supported Queries

## Overview
The chatbot in Invisor.ai is designed to summarize and explain existing
machine learning outputs. It does not perform new analysis, predictions,
or reasoning.

---

## Supported Query Categories

### 1. Churn-Related Queries
- Overall churn risk summaries
- High-risk vs low-risk counts
- Global churn-driving features
- Customer-level churn explanations (when selected)

### 2. Customer Segmentation Queries
- Segment definitions
- Segment characteristics
- Segment distribution and counts

### 3. Dataset-Level Insights
- Dataset size and structure
- Aggregate trends across churn and segments

---

## Unsupported Queries
The chatbot does not support:
- Model reconfiguration or retraining
- What-if or scenario analysis
- Causal explanations
- Prescriptive business advice
- Open-ended AI conversations

---

## Design Principle
All chatbot responses must be derived from existing ML outputs or
precomputed summaries. No new inference or reasoning is performed.
